Приложение для многокритериального выбора альтернатив с использованием правил нечеткого вывода, написано на J2SE (Swing).
Для корректной работы необходимо подключение объектов, описывающих правила вывода (наследующие класс Function, подключаются уже собранные .class файлы в требуемом порядке).
